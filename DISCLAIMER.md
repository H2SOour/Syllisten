## 4. Third-party processing

Rule-based segmentation does not intentionally send text to OpenAI, but text is still processed by the server hosting Syllisten.

When the user selects OpenAI mode, submitted text is sent to OpenAI solely using the API key entered by that user for the current session. Syllisten does not include a developer OpenAI API key and is not intended to charge the developer's OpenAI account.

Speech is generated primarily through `edge-tts`, an unofficial client for Microsoft Edge online speech synthesis. Text selected for speech may therefore be transmitted to Microsoft-operated online infrastructure. Browser speech is used as a fallback and may itself use local or remote processing depending on the user's environment.

Hosting providers, content-delivery networks, reverse proxies, analytics systems, browser extensions, operating systems, and network operators may process technical data outside the developer's direct control.
