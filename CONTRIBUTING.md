# Contributing

Before contributing:

1. Do not include third-party text, audio, images, fonts, models, datasets, credentials, or code without documented permission and license.
2. Never commit API keys, passwords, tokens, personal data, or confidential information.
3. Keep user-facing changes synchronized across all five interface languages.
4. Test rule mode and any affected user-key API behavior.
5. Preserve all copyright, license, and attribution notices.

Unless clearly stated otherwise in writing, contributions intentionally submitted for inclusion are offered under the same MIT License as Syllisten, and contributors represent that they have the right to submit them.
